## 数据结构-列表（list）

列表是写在方括号 `[]` 之间、用逗号分隔开的元素列表

列表中元素的类型可以不相同，它支持数字，字符串甚至可以包含列表（所谓嵌套）。

**列表截取**

语法格式为：`list[start:end]` ，其中 start（包含）是切片开始的索引，end（不包含）是切片结束的索引。

列表的切片可以加上步长参数 step，语法格式如下：`list[start:end:step]` 。

**字符串拼接与重复**

加号 `+` 是列表连接运算符，星号 `*` 是重复操作。

**注意**：与Python字符串不一样的是，列表中的元素是可以改变的。

## 数据结构-字典（dict）

字典是一种映射类型，字典用 `{ }` 标识，它是一个无序的 `键(key) : 值(value)` 的集合。

在同一个字典中，键(key)必须是唯一的，且是不可变类型。

## 数据结构-集合（set）

集合（Set）是一种无序、可变的数据类型，用于存储唯一的元素。

在 Python 中，集合使用大括号 `{}` 表示，元素之间用逗号 , 分隔。
