## 数据结构-列表（list）

列表是 Python 中最常用的**容器类型**，可以存储**任意数量、任意类型**的数据。

类比：列表就像一个**购物车**，可以把各种东西放进去。

### 创建列表

| 方式          | 示例                            | 说明                     |
| :------------ | :------------------------------ | :----------------------- |
| 方括号        | `[1, 2, 3]`                     | 最常用                   |
| `list()` 函数 | `list("abc")` → `['a','b','c']` | 将其他类型转为列表       |
| 列表推导式    | `[x*2 for x in range(5)]`       | 高级创建方式（后面会讲） |

例子：

```python
# 各种创建方式
fruits = ["苹果", "香蕉"]           # 方括号
chars = list("Python")              # ['P','y','t','h','o','n']
numbers = list(range(5))            # [0,1,2,3,4]
```

注：字典可以包含字典（实现复杂数据结构）。

### 访问列表元素

和字符串一样，列表也支持**索引**和**切片**：

```python
fruits = ["苹果", "香蕉", "橙子", "葡萄", "西瓜"]

# 正向索引（从0开始）
print(fruits[0])    # 苹果
print(fruits[2])    # 橙子

# 负向索引（从-1开始）
print(fruits[-1])   # 西瓜
print(fruits[-3])   # 橙子

# 切片 [start:end:step]
print(fruits[1:4])   # ['香蕉', '橙子', '葡萄']（索引1到3）
print(fruits[:3])    # ['苹果', '香蕉', '橙子']
print(fruits[::2])   # ['苹果', '橙子', '西瓜']（步长2）
print(fruits[::-1])  # ['西瓜', '葡萄', '橙子', '香蕉', '苹果']（反转）
```

### 修改列表（可变性）

列表是**可变**的，可以增删改元素（字符串不可变，列表可以变）

```python
fruits = ["苹果", "香蕉", "橙子"]

# 修改元素
fruits[1] = "草莓"
print(fruits)   # ['苹果', '草莓', '橙子']

# 添加元素
fruits.append("葡萄")        # 在末尾添加
print(fruits)   # ['苹果', '草莓', '橙子', '葡萄']

fruits.insert(1, "芒果")     # 在索引1位置插入
print(fruits)   # ['苹果', '芒果', '草莓', '橙子', '葡萄']

# 删除元素
del fruits[2]               # 删除索引2的元素
print(fruits)   # ['苹果', '芒果', '橙子', '葡萄']

removed = fruits.pop()      # 删除并返回最后一个元素
print(removed)  # 葡萄
print(fruits)   # ['苹果', '芒果', '橙子']

fruits.remove("芒果")        # 删除指定值（若有重复，只删除第一个）
print(fruits)   # ['苹果', '橙子']
```

### 列表常用方法

| 方法           | 作用                  | 示例                                |
| :------------- | :-------------------- | :---------------------------------- |
| `append(x)`    | 末尾添加元素          | `[1,2].append(3)` → `[1,2,3]`       |
| `insert(i, x)` | 指定位置插入          | `[1,3].insert(1,2)` → `[1,2,3]`     |
| `extend(iter)` | 合并另一个列表        | `[1,2].extend([3,4])` → `[1,2,3,4]` |
| `pop(i)`       | 删除并返回索引i的元素 | `[1,2,3].pop(1)` → `2`              |
| `remove(x)`    | 删除第一个值为x的元素 | `[1,2,1].remove(1)` → `[2,1]`       |
| `index(x)`     | 返回第一个x的索引     | `[1,2,3].index(2)` → `1`            |
| `count(x)`     | 统计x出现次数         | `[1,1,2].count(1)` → `2`            |
| `sort()`       | 排序（原地修改）      | `[3,1,2].sort()` → `[1,2,3]`        |
| `reverse()`    | 反转（原地修改）      | `[1,2,3].reverse()` → `[3,2,1]`     |
| `clear()`      | 清空列表              | `[1,2].clear()` → `[]`              |

例子：

```python
# 综合示例
numbers = [3, 1, 4, 1, 5, 9, 2]

# 统计
print(numbers.count(1))       # 2
print(numbers.index(9))       # 5

# 排序（原地修改）
numbers.sort()
print(numbers)                # [1, 1, 2, 3, 4, 5, 9]

# 反转
numbers.reverse()
print(numbers)                # [9, 5, 4, 3, 2, 1, 1]

# 合并
numbers.extend([10, 11])
print(numbers)                # [9, 5, 4, 3, 2, 1, 1, 10, 11]
```

### 列表常用操作

#### 1. 判断元素是否存在

通过 `in` ：

```python
fruits = ["苹果", "香蕉", "橙子"]

print("苹果" in fruits)    # True
print("葡萄" in fruits)    # False
print("西瓜" not in fruits) # True
```

#### 2. 列表合并

```python
a = [1, 2, 3]
b = [4, 5, 6]

# 方式1：使用 +（创建新列表）
c = a + b
print(c)   # [1, 2, 3, 4, 5, 6]

# 方式2：使用 extend（修改原列表）
a.extend(b)
print(a)   # [1, 2, 3, 4, 5, 6]
```

#### 3. 列表复制

```python
original = [1, 2, 3]

# ❌ 错误：直接赋值（指向同一个对象）
copy1 = original
copy1[0] = 999
print(original)   # [999, 2, 3] ← 原列表也被改了！

# ✅ 正确：浅拷贝
copy2 = original.copy()      # 方法1
copy3 = original[:]          # 方法2
copy4 = list(original)       # 方法3
```

补充解释：

- Python 中“变量名”只是一个“标签”或者“指针”，它指向内存中的储存对象。
- 赋值（`=`）就是把“变量名”作为“标签”贴到或指向储存对象。
- 如 `a=10,b=a` 就是让变量 `a` 指向 `10` ，然后让 `b` 指向当前 `a` 所指向的对象。
- 如果（可变）对象改变了，则 `a` 和 `b` 均一起改变。
- 如果是不可变对象，那么 `b` 只能通过指向其他对象来改变，此时 `a` 不变。

#### 4. 列表推导式（强大技巧）

```python
# 生成 1-10 的平方列表
squares = [x**2 for x in range(1, 11)]
print(squares)   # [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

# 带条件的推导式（只取偶数）
evens = [x for x in range(1, 11) if x % 2 == 0]
print(evens)     # [2, 4, 6, 8, 10]

# 等价于：
evens = []
for x in range(1, 11):
    if x % 2 == 0:
        evens.append(x)
```



## 数据结构-元组（tuple）

元组（Tuple）是 Python 中另一种重要的容器类型。**元组是不可变的列表**——一旦创建，就不能修改。

### 创建元组

元组用**圆括号 `()`** 表示，可以存储任意类型的数据：

```python
# 创建元组
t1 = (1, 2, 3)              # 整数元组
t2 = ("a", "b", "c")        # 字符串元组
t3 = (1, "hello", 3.14)     # 混合类型
t4 = ()                      # 空元组
t = 1, 2, 3
print(type(t))  # <class 'tuple'>  ← 没括号也是元组！

# 特别注意：单元素元组要加逗号
t5 = (5,)      # 正确：这是元组
not_tuple = (5)  # 错误！这是整数 5，括号被当作数学优先级
```

### 元组的操作

由于元组不可变，**没有** `append()`、`remove()`、`sort()` 等方法。只支持索引访问、切片等。

```python
t = (1, 2, 3, 4, 5)

# ✅ 支持的操作
print(t[0])          # 1（索引访问）
print(t[1:3])        # (2, 3)（切片）
print(len(t))        # 5（长度）
print(3 in t)        # True（成员检查）
```

### 元组和列表的互相转换

```python
# 列表 → 元组
my_list = [1, 2, 3]
my_tuple = tuple(my_list)
print(my_tuple)      # (1, 2, 3)

# 元组 → 列表
my_tuple = (1, 2, 3)
my_list = list(my_tuple)
print(my_list)       # [1, 2, 3]
```

### 元组的"可变"陷阱

虽然元组不可变，但如果元组**包含可变对象**（如列表），这个列表本身是可以修改的。

元组的"不可变"是指不能替换其中的元素引用，但引用的对象本身可以变。

```python
t = (1, 2, [3, 4])    # 元组包含一个列表

# t[2] = [5, 6]       # ❌ 报错：不能替换元组中的元素
t[2][0] = 999         # ✅ 可以修改列表内部（列表本身没有变）
print(t)              # (1, 2, [999, 4])

# 结论：元组的"不可变"是指不能替换其中的元素引用，但引用的对象本身可以变
```



## 数据结构-字典（dict）

字典是以**键值对（key-value）**的形式存储数据的数据结构。

类比：像一本实际的字典——通过"词"（键）查找"释义"（值）。

### 创建字典

字典用花括号 `{}` 表示，每个元素都是一个键值对，用冒号 `:` 连接。

**创建一个字典：**

```python
person = {
    "name": "张三",    # 键: "name"，值: "张三"
    "age": 18,         # 键: "age"，值: 18
    "city": "北京"     # 键: "city"，值: "北京"
}
```

**访问字典：**

```
print(person["name"])   # 张三
print(person["age"])    # 18
```

**空字典：**

```python
empty_dict = {}
```

### 字典的核心特点

| 特点             | 说明                                                         |
| :--------------- | :----------------------------------------------------------- |
| **键值对存储**   | 每个键对应一个值，通过键访问值                               |
| **键唯一**       | 同一个字典中，键不能重复（后面的会覆盖前面的）               |
| **键必须不可变** | 键可以是字符串、数字、元组，但不能是列表                     |
| **值任意类型**   | 值可以是任何 Python 对象（数字、字符串、列表、甚至另一个字典） |
| **可变**         | 可以增、删、改键值对                                         |
| **无序**         | Python 3.6 之前无序，3.7+ 保持插入顺序                       |

### 字典的增删改查

- 查询（访问）
- 添加/修改
- 删除
- 清空

```python
person = {"name": "张三", "age": 18}

# 1. 查询（访问）
print(person["name"])         # 张三
print(person.get("city"))     # None（不存在时返回 None，不报错）
print(person.get("city", "未知"))  # 未知（可指定默认值）

# 2. 添加/修改
person["city"] = "北京"        # 新增键值对
person["age"] = 19            # 修改已有键的值
print(person)                 # {'name': '张三', 'age': 19, 'city': '北京'}

# 3. 删除
del person["city"]            # 删除键值对
age = person.pop("age")       # 删除并返回被删除的值
print(age)                    # 19

# 4. 清空
person.clear()                # 清空字典
```

### 字典的常用方法

| 方法                | 作用                               | 示例                                                         |
| :------------------ | :--------------------------------- | :----------------------------------------------------------- |
| `keys()`            | 查看所有键（视图对象）             | `person.keys()` → `dict_keys(['name', 'age'])`               |
| `values()`          | 查看所有值（视图对象）             | `person.values()` → `dict_values(['张三', 18])`              |
| `items()`           | 查看所有键值对（视图对象）         | `person.items()` → `dict_items([('name', '张三'), ('age', 18)])` |
| `get(key, default)` | 获取值，不存在返回默认值           | `person.get("city", "未知")`                                 |
| `pop(key)`          | 删除指定的键值对，并返回被删除的值 | `person.pop("age")`                                          |
| `update(dict2)`     | 合并另一个字典                     | `person.update({"city": "北京"})`                            |

### 遍历字典

- 遍历键
- 遍历值
- 遍历键值对

```python
person = {"name": "张三", "age": 18, "city": "北京"}

# 1. 遍历键
for key in person:
    print(key)        # name, age, city

# 2. 遍历值
for value in person.values():
    print(value)      # 张三, 18, 北京

# 3. 遍历键值对（推荐）
for key, value in person.items():
    print(f"{key}: {value}")
# name: 张三
# age: 18
# city: 北京
```

## 数据结构-集合（set）

**集合**是一个**无序**、**元素唯一**（自动去重）的容器，专门用于**数学上的集合运算**（并集、交集、差集等）。

### 集合的创建

在 Python 中，集合使用大括号 `{}` 表示，元素之间用逗号 , 分隔。

```python
# 1. 花括号（最常用）
fruits = {"苹果", "香蕉", "橙子"}

# 2. 从列表转换（重复元素会被自动去掉），注意顺序可能不同
numbers = set([1, 2, 2, 3, 3, 3])   # {1, 2, 3}

# 3. 从字符串转换（字符去重）
chars = set("hello")                # {'h', 'e', 'l', 'o'}

# 4. 集合推导式
squares = {x**2 for x in range(5)}  # {0, 1, 4, 9, 16}

# 5. 空集合（）
empty_set = set()
```

**注意：**`{}` 是空字典，空集合必须用 `set()` 

**核心特性**：

- **元素唯一**：自动去重，重复的元素只保留一个
- **无序**：不能通过索引访问（没有 `s[0]`）
- **可变**：可以增删元素
- **元素必须可哈希**：只能放不可变类型（数字、字符串、元组），不能放列表或字典

### 集合的常用操作

#### 1. 添加和删除元素

- 添加元素
- 删除元素
- 随机删除
- 清空集合

```python
s = {1, 2, 3}

# 添加元素
s.add(4)
print(s)          # {1, 2, 3, 4}

# 删除元素
s.remove(2)       # 删除元素 2（不存在会报错 KeyError）
print(s)          # {1, 3, 4}

s.discard(5)      # 删除元素 5（不存在不报错，安全）
print(s)          # {1, 3, 4}

# 随机删除并返回一个元素（因为无序，所以是随机的）
popped = s.pop()
print(popped)     # 1 或 3 或 4（随机）

# 清空集合
s.clear()
print(s)          # set()
```

#### 2. 集合运算（核心用途）

- 并集
- 交集
- 差集
- 对称差集
- 判断子集/超集

```python
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

# 并集（A 或 B 中的所有元素）
print(A | B)          # {1, 2, 3, 4, 5, 6}
print(A.union(B))     # 同上

# 交集（A 和 B 共有的元素）
print(A & B)          # {3, 4}
print(A.intersection(B))  # 同上

# 差集（在 A 中但不在 B 中的元素）
print(A - B)          # {1, 2}
print(A.difference(B))  # 同上

# 对称差集（不同时在 A 和 B 中的元素）
print(A ^ B)          # {1, 2, 5, 6}
print(A.symmetric_difference(B))  # 同上

# 判断子集/超集
C = {1, 2}
print(C.issubset(A))   # True（C 是 A 的子集）
print(A.issuperset(C)) # True（A 是 C 的超集）
print(A.isdisjoint(B)) # False（A 和 B 有交集）
```

### 集合 vs 列表（对比）

| 特性         | 列表 (list)        | 集合 (set)         |
| :----------- | :----------------- | :----------------- |
| **顺序**     | 有序（按插入顺序） | 无序               |
| **重复**     | 允许重复           | 自动去重           |
| **索引**     | 支持 `[0]`         | 不支持             |
| **可变**     | ✅ 可变             | ✅ 可变             |
| **数学运算** | 不支持             | 支持（并、交、差） |
| **查找速度** | O(n) 较慢          | O(1) 极快          |



## 容器和数据结构

### 容器 vs 数据结构

| 概念                          | 含义                                                         | 例子                                 |
| :---------------------------- | :----------------------------------------------------------- | :----------------------------------- |
| **容器 (Container)**          | 专门指**存储多个元素**的对象，强调“能装东西”                 | 列表、元组、字典、集合、字符串       |
| **数据结构 (Data Structure)** | 更宽泛的概念，指数据在计算机中的**组织方式**，不仅包括存储，还包括操作逻辑 | 列表、栈、队列、链表、树、图、哈希表 |

**关系**：**容器是数据结构的一个子集**。

```reStructuredText
数据结构（大概念）
├── 容器（存储数据）
│   ├── 列表
│   ├── 元组
│   ├── 字典
│   ├── 字符串
│   └── 集合
├── 抽象数据结构（强调操作逻辑）
│   ├── 栈（LIFO）
│   ├── 队列（FIFO）
│   └── 堆（优先队列）
└── 其他
    ├── 链表
    ├── 树
    └── 图
```

注：在 Python 官方文档中，**容器**有严格定义，是指支持 `in` 操作符的对象，用于判断某个元素是否在其中。

### 不同容器的区别

| 特性     | 列表 (list)  | 元组 (tuple) | 字典 (dict)    | 集合 (set)     |
| :------- | :----------- | :----------- | :------------- | :------------- |
| 语法     | `[]`         | `()`         | `{}`           | `{}`           |
| 顺序     | 有序         | 有序         | 有序 (3.7+)    | 无序           |
| 可变     | ✅            | ❌            | ✅              | ✅              |
| 元素形式 | 值           | 值           | 键值对         | 值             |
| 访问方式 | 索引 (0,1,2) | 索引 (0,1,2) | 键 (key)       | 无（不能索引） |
| 重复元素 | ✅            | ✅            | 键不可重复     | ❌（自动去重）  |
| 使用场景 | 有序序列     | 不可变序列   | 快速查找、映射 | 去重、集合运算 |

## 序列类型的通用操作

在讲完列表、元组、字符串之后，你会发现它们有很多**共同的操作**。这一节就是把这些通用操作归纳总结，建立“序列”这个整体概念。

### 什么是序列

**序列（Sequence）** 是 Python 中一大类容器的统称，核心特征是：**有序、可以通过索引访问、支持切片** 。

例子：

```python
# 这些都是序列
"hello"          # 字符串
[1, 2, 3]        # 列表
(1, 2, 3)        # 元组
range(5)         # 范围对象

# 这些不是序列
{1, 2, 3}        # 集合（无序，无索引）
{"a": 1}         # 字典（映射，键访问）
```

### 序列的通用操作一览

以下操作对**列表、元组、字符串**都适用：

| 操作        | 说明         | 列表示例          | 元组示例          | 字符串示例             |
| :---------- | :----------- | :---------------- | :---------------- | :--------------------- |
| `索引`      | 通过位置访问 | `[0]`             | `[0]`             | `[0]`                  |
| `切片`      | 截取子序列   | `[1:3]`           | `[1:3]`           | `[1:3]`                |
| `拼接`      | 合并两个序列 | `+`               | `+`               | `+`                    |
| `重复`      | 重复 n 次    | `* 3`             | `* 3`             | `* 3`                  |
| `成员`      | 判断是否存在 | `in` / `not in`   | `in` / `not in`   | `in` / `not in`        |
| `长度`      | 元素个数     | `len()`           | `len()`           | `len()`                |
| `最大/最小` | 极值         | `max()` / `min()` | `max()` / `min()` | `max()` / `min()`      |
| `遍历`      | 逐个访问     | `for x in list:`  | `for x in tuple:` | `for x in str:`        |
| `查找`      | 返回索引     | `.index()`        | `.index()`        | `.find()` / `.index()` |
| `计数`      | 统计次数     | `.count()`        | `.count()`        | `.count()`             |

注意：序列拼接 `+`：只能同类型拼接。

例子：

```python
# ========== 1. 索引 ==========
s1 = "hello"
s2 = [10, 20, 30]
s3 = (100, 200, 300)

print(s1[0])    # h
print(s2[-1])   # 30
print(s3[1])    # 200


# ========== 2. 切片 ==========
```

### 序列的分类

根据**可变性**，序列分为两类：

## 函数

函数是“接收输入，处理逻辑，返回输出”的代码块。

### 函数的组成部分

```python
def add(a, b):          # 1. 函数头：def + 函数名 + 参数
    """返回两个数的和"""  # 2. 文档字符串（说明书）
    result = a + b      # 3. 函数体（执行的代码）
    return result       # 4. 返回值
```

| 组成部分   | 说明                                    | 是否必须              |
| :--------- | :-------------------------------------- | :-------------------- |
| 函数头     | `def` 关键字 + 函数名 + 参数列表 + 冒号 | ✅ 必须                |
| 文档字符串 | 用三引号写的说明文字                    | ❌ 建议有              |
| 函数体     | 缩进的代码块                            | ✅ 必须（至少 `pass`） |
| `return`   | 返回结果                                | ❌ 没有则返回 `None`   |

### 函数的定义与调用

**1.无参数的函数**

```python
# 定义函数
def say_hello():
    print("Hello!")

# 调用函数
say_hello()   # Hello!
```

**2.带参数的函数**

```python
def greet(name):
    print(f"你好，{name}！")

greet("张三")   # 你好，张三！
greet("李四")   # 你好，李四！
```

**3.带返回值的函数**

```python
def add(a, b):
    return a + b

result = add(3, 5)
print(result)   # 8
```

### 参数详解

参数的本质是：把外部的数据，赋值给函数内部的临时变量。

| 数类型            | 适用场景                                           | 核心规则                                                     |
| :---------------- | :------------------------------------------------- | :----------------------------------------------------------- |
| **1. 位置参数**   | **最常用**。传几个值，它就按顺序接收几个值。       | 传参的**顺序**和**数量**必须严格一一对应，多一个少一个都报错。 |
| **2. 默认参数**   | 默认情况（即为输入该参数下）使用一个默认值。       | 定义时赋值 `def f(a, b=10)`。 **注意**：默认参数必须放在位置参数的**后面**。 |
| **3. 关键字参数** | 调用函数时，怕记错顺序（比如三个参数都是数字）。   | 调用时指名道姓：`f(a=1, c=3, b=2)`，**顺序可以随便打乱**。   |
| **4. 可变参数**   | 不确定用户会传几个值。比如计算任意多个数字的总和。 | `*args` 接收一堆位置参数（打包成元组），`**kwargs` 接收一堆关键字参数（打包成字典）。 |

#### 1. 位置参数（最常用）

```python
def introduce(name, age, city):
    print(f"我叫{name}，{age}岁，来自{city}")

# 调用时按位置传入
introduce("张三", 18, "北京")
# 我叫张三，18岁，来自北京
```

#### 2. 默认参数

```
def greet(name, greeting="你好"):
    print(f"{greeting}，{name}！")

greet("张三")            # 你好，张三！
greet("李四", "Hello")   # Hello，李四！
```

**注意**：默认参数必须放在位置参数**后面**。

#### 3. 关键字参数（调用时指定参数名）

```python
def introduce(name, age, city):
    print(f"{name}，{age}岁，来自{city}")

# 调用时可以打乱顺序
introduce(city="北京", name="张三", age=18)
# 张三，18岁，来自北京
```

#### 4. 可变参数

```python
# *args：接收任意数量的位置参数（打包成元组）
def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3))       # 6
print(sum_all(10, 20, 30, 40)) # 100


# **kwargs：接收任意数量的关键字参数（打包成字典）
def show_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

show_info(name="张三", age=18, city="北京")
# name: 张三
# age: 18
# city: 北京
```

### 作用域

函数内创建的变量是局部变量，不会影响函数外的变量（即使同名），并且出了函数就变量就消亡。

如果想影响函数外的变量，或者访问函数外的变量，则需要使用 `global` 获取外部的全局变量。

```python
count = 0

def increment():
    global count   # 声明要修改全局变量
    count += 1

increment()
print(count)   # 1
```

### `pass` 占位符

`pass` 是 Python 中的**空操作语句**，它的作用是**占位**——即"这里需要写代码，但暂时什么都不做"。

只要 Python 语法要求必须有代码块的地方，而你暂时不想写具体逻辑，都可以用 `pass`。

当函数体还没想好怎么写时，用 `pass` 占位，避免语法错误：

```python
def future_function():
    pass   # 以后再来实现
```

### lambda 表达式

**lambda** 是 Python 中的**匿名函数**——即没有函数名的函数。

它的核心用途是：当你需要一个简单函数，但不想用 `def` 去正式定义一个完整函数时，用 lambda 可以一行写完。

#### 基本语法

```python
lambda 参数列表: 表达式
```

- **参数列表**：和普通函数一样，可以有 0 个、1 个或多个参数
- **冒号 `:`**：分隔参数和表达式
- **表达式**：只能写一个表达式，其计算结果就是返回值（**不能写多条语句**）

例子：

```python
# 用 def 定义（常规写法）
def add(a, b):
    return a + b

# 用 lambda 定义（匿名函数）
lambda a, b: a + b

# 无参数
lambda: "Hello"

# 单个参数
lambda x: x * 2

# 多个参数
lambda x, y: x + y

# 默认参数
lambda x, y=10: x + y

# 可变参数
lambda *args: sum(args)
```

#### 基本使用

**1. 赋值给变量（不常用）**

```python
add = lambda x, y: x + y
print(add(3, 5))   # 8

square = lambda x: x ** 2
print(square(4))   # 16
```

**注意**：这种用法是把 lambda 被赋值给变量，它本质上就相当于一个普通函数，失去了“匿名”的意义。

**2. 作为参数传递给高阶函数（lambda 的真正用途）**

例子：

```python
students = [
    ("张三", 85),
    ("李四", 92),
    ("王五", 78),
]

# 按成绩排序（取元组的第二个元素作为排序依据）
sorted_students = sorted(students, key=lambda x: x[1], reverse=True)
print(sorted_students)  # [('李四', 92), ('张三', 85), ('王五', 78)]
```

### lambda vs def

| 对比维度     | def                | lambda                   |
| :----------- | :----------------- | :----------------------- |
| **名称**     | 必须有函数名       | 匿名（无函数名）         |
| **语法**     | 多行，支持多个语句 | 单行，只能有一个表达式   |
| **返回值**   | `return` 显式返回  | 自动返回表达式的计算结果 |
| **适用场景** | 复杂逻辑、多行代码 | 简单逻辑、一次性使用     |
| **可读性**   | 适合长代码         | 适合短代码               |
