# 语法基础

## 程序结构

一个 Python 程序的基本结构包括：**缩进、注释、语句与表达式、函数与类定义**等核心要素。

### 编码

默认情况下，Python3 源码文件以 UTF-8 编码，所有字符串都是 unicode 字符串。

### 缩进

Python **使用缩进来表示代码块**，而不是像其他语言（如 C、Java）使用花括号 `{}`。

缩进的空格数可以自行改变，但是同一个代码块的语句必须具有相同的缩进空格数。

```python
# 正确写法：同一代码块缩进一致
if True:
    print("这是缩进4个空格")
    print("同一代码块，缩进相同")
    
# 错误写法：缩进不一致会报错
if True:
    print("正确缩进")
   print("缩进不一致，IndentationError")  # 报错！
```

**规范**：推荐使用 4个空格 作为一级缩进，不要混用 Tab 和空格。

### 注释

注释不会被程序执行，用于解释代码。

| 类型     | 写法                  | 示例                 |
| :------- | :-------------------- | :------------------- |
| 单行注释 | `#` 开头              | `# 这是一个注释`     |
| 多行注释 | 三引号 `'''` 或 `"""` | `'''多行注释内容'''` |

### 导入模块

在 python 用 `import` 或者 `from...import` 来导入相应的模块。

**语法**：

1. 将整个XXX模块导入，格式为： `import XXX` ，其中 `XXX` 是模块名。
2. 从 XXX 模块中导入 Y1 函数，格式为：`from XXX import Y1` ，其中 `Y1` 是函数名。
3. 从 XXX 模块中导入 Y1、Y2、Y3 函数，格式为：`from XXX import Y1, Y2, Y3` 。
4. 将 XXX 模块中的全部函数导入，格式为： `from XXX import * ` 。

注意：第一种用法中，引用模块中的 Y1 函数时需要加上模块名： `XXX.Y1` ；而后面几种用法中则不需要加上：`Y1` 。

### 代码组织层级

一个完整的 Python 程序通常按以下层级组织：

```python
# 1. 模块头部注释（可选）
#!/usr/bin/env python3   # Shebang（Linux/macOS）
# -*- coding: utf-8 -*-   # 编码声明

# 2. 导入模块
import sys
from math import pi

# 3. 全局常量
MAX_SIZE = 100

# 4. 函数定义
def calculate_area(radius):
    return pi * radius ** 2

# 5. 类定义
class Circle:
    def __init__(self, radius):
        self.radius = radius

# 6. 主程序入口
if __name__ == "__main__":
    print("程序开始执行")
    result = calculate_area(5)
    print(f"面积: {result}")
```



### 行连接与多行语句

Python 通常是一行写完一条语句，但如果语句很长，我们可以使用反斜杠 `\` 来实现多行语句。

如果在 `[]`, `{}`, 或 `()` 中的多行语句，则不需要使用反斜杠 `\` 。

```python
# 使用反斜杠 \ 续行
total = 1 + 2 + 3 + \
        4 + 5

# 括号内的元素可自动换行
numbers = [1, 2, 3, 
           4, 5, 6]

result = (1 + 2 + 3 + 
          4 + 5 + 6)
```

### 多条语句写在一行

Python 可以在同一行中使用多条语句，语句之间使用分号 `;` 分割。

```python
x = 1; y = 2; print(x + y)  # 可用分号分隔，但降低可读性
```



## 变量

变量是**存储数据的容器**，可以理解为给数据贴上的标签。

Python 中的变量不需要声明。每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。

在 Python 中，变量就是变量，它没有类型，我们所说的“类型”是变量所指的内存中对象的类型。

#### 变量命名规则

| 规则                     | 说明                     | 正确示例       | 错误示例 |
| :----------------------- | :----------------------- | :------------- | :------- |
| 由字母、数字、下划线组成 | 不能包含特殊符号         | `name_1`       | `name@1` |
| 不能以数字开头           | 首字符必须是字母或下划线 | `var1`、`_var` | `1var`   |
| 区分大小写               | `Name` 和 `name` 不同    | `age` ≠ `Age`  | —        |
| 不能使用关键字           | Python 保留字            | `my_if`        | `if`     |

### 命名规范（PEP 8 推荐）

```python
# 变量名：小写字母 + 下划线（蛇形命名法）
user_name = "张三"
total_score = 100

# 常量名：全大写 + 下划线
MAX_HEIGHT = 180
PI = 3.14159

# 函数名，动词+名词
def calculate_area():
	pass

# 类名：大驼峰（首字母大写）
class UserAccount:
    pass

# 下划线开头的通常表示“内部使用”或“私有”变量
_total = 100     

# 双下划线开头，有特殊含义
__private_var
```

### 变量赋值

```python
# 基本赋值
x = 10

# 链式赋值（多个变量赋同一值）
a = b = c = 0

# 平行赋值（交换变量值）
m, n = 5, 10
m, n = n, m      # 交换：m=5, n=10 to m=10, n=5
```



## 数据类型

Python 中**变量没有类型，对象才有类型**。赋值后，变量指向对应类型的对象。

### 基本数据类型一览

| 类型分类   | 具体类型   | 示例                  | 说明             |
| :--------- | :--------- | :-------------------- | :--------------- |
| **数字**   | `int`      | `42`、`-5`、`0`       | 整数，长度无限制 |
|            | `float`    | `3.14`、`-0.5`、`2e3` | 浮点数（双精度） |
|            | `complex`  | `3+4j`                | 复数             |
| **布尔**   | `bool`     | `True`、`False`       | 注意首字母大写   |
| **字符串** | `str`      | `"Hello"`、`'Python'` | 单引号或双引号   |
| **空值**   | `NoneType` | `None`                | 表示“无”         |

### 数字类型详解

python中数字有四种类型：整数、布尔型、浮点数和复数。

```python
# 整数 int
age = 25
big_num = 10 ** 100   # Python支持大整数

# 浮点数 float
pi = 3.14159
score = 98.5
f_num = 3e-2

# 布尔值（本质是int的子类，True=1，False=0）
is_student = True
is_working = False
print(True + True)    # 2
print(True + False)   # 1

# complex (复数) 由实部和虚部组成
comp_num = 2 + 4j
comp_num = complex(2,4)
```



### 字符串类型详解

这里只是简单介绍字符串类型，后面还会专门介绍字符串。

**字符串定义方式**

```python
# 单引号
str1 = 'Hello'

# 双引号（与单引号完全等价）
str2 = "World"

# 三引号（多行字符串）
str3 = """第一行
第二行
第三行"""

# 转义字符
path = "C:\\Users\\name"      # \ 需要转义
path_raw = r"C:\Users\name"   # 原始字符串（r前缀）
```

**字符串基本操作**

```python
# 字符串拼接
first = "Hello"
last = "Python"
msg = first + " " + last    # "Hello Python"

# 字符串重复
line = "-" * 10             # "----------"

# 字符串格式化
name = "小明"
age = 18

# 方式1：f-string（Python 3.6+，推荐）
print(f"我叫{name}，今年{age}岁")

# 方式2：format() 方法
print("我叫{}，今年{}岁".format(name, age))

# 方式3：% 格式化（旧式）
print("我叫%s，今年%d岁" % (name, age))
```

**注意**：

- 字符串中的反斜杠 `\` 是用来转义的，在字符串前面添加一个 `r` 可以让反斜杠不发生转义。 如 `r"line \n"` 则 `\n` 会显示，而不是换行。
- Python 中的字符串不能改变，不能向字符串的一个索引位置赋值。

### 类型检查

type() 函数：查询变量所指的对象类型。

```python
# type() 返回类型对象
x = 10
print(type(x))        # <class 'int'>

# isinstance() 判断是否为某类型（推荐）
print(isinstance(x, int))      # True
print(isinstance(x, (int, float)))  # True（元组中任一匹配即可）

# 动态改变类型
value = 100
print(type(value))    # int
value = "hello"
print(type(value))    # str
```

### 数据类型转换

 我们使用 int()、float()、str() 等预定义函数将对象的数据类型显示转换为所需的数据类型。

- `int()` 强制转换为整型。
- `float()` 强制转换为浮点型。
- `str()` 强制转换为字符串类型。

```python
int("123")        # 123，字符串转整数
float(10)         # 10.0
str(3.14)         # "3.14"
bool(0)           # False（0、0.0、空字符串、None 都为 False）
bool(5)           # True
```

**注意**：在 Python 中，所有非零的数字和非空的字符串、列表、元组等数据类型都被视为 True，只有 0、空字符串、空列表、空元组等被视为 False。因此，在进行布尔类型转换时，需要注意数据类型的真假性。

## 基本输入输出

输入输出是程序与用户交互的桥梁。

Python 通过 `print()` 输出数据，通过 `input()` 获取用户输入。

### print() 输出函数

**基本用法**

```python
# 输出单个值
print("Hello World")

# 输出多个值（用空格分隔）
print("姓名:", "张三", "年龄:", 25)
# 输出：姓名: 张三 年龄: 25

# 输出任意类型
print(42)
print(3.14)
print(True)
print([1, 2, 3])
```

**print() 参数详解**

print() 可以设置的参数的有四个：

| 参数    | 作用               | 默认值               |
| :------ | :----------------- | :------------------- |
| `sep`   | 多个值之间的分隔符 | 空格 `" "`           |
| `end`   | 输出结尾的字符     | 换行 `"\n"`          |
| `file`  | 输出目标（文件等） | `sys.stdout`（屏幕） |
| `flush` | 是否立即刷新缓冲区 | `False`              |

例子：

```python
# 修改分隔符 sep
print("苹果", "香蕉", "橙子", sep=", ")
# 输出：苹果, 香蕉, 橙子

print("A", "B", "C", sep="-")
# 输出：A-B-C

# 修改结尾 end
print("加载中", end="")
print(".", end="")
print(".", end="")
print("完成")
# 输出：加载中...完成

# 组合使用
print("姓名", "年龄", "城市", sep=" | ", end="\n\n")
print("张三", "25", "北京", sep=" | ")
# 输出：
# 姓名 | 年龄 | 城市
#
# 张三 | 25 | 北京
```

### 格式化输出

三种方式：

```python
name = "李华"
age = 18
score = 92.5

# 1. f-string（Python 3.6+，推荐）
print(f"姓名：{name}，年龄：{age}，成绩：{score}")

# 2. 格式控制（保留小数）
pi = 3.1415926
print(f"π ≈ {pi:.2f}")        # π ≈ 3.14
print(f"百分比：{0.856:.1%}")   # 百分比：85.6%

# 3. 宽度对齐
print(f"|{name:<10}|")   # 左对齐，宽度10   |李华        |
print(f"|{name:>10}|")   # 右对齐           |        李华|
print(f"|{name:^10}|")   # 居中对齐         |    李华    |
```

## 运算符

**表达式**是由变量、常量和运算符组成的式子，**运算符**是用于执行操作的符号。

Python 提供了丰富的运算符类型。

### 算术运算符

用于数学计算，最基础的运算符。

| 运算符 | 名称         | 示例      | 结果       |
| :----- | :----------- | :-------- | :--------- |
| `+`    | 加法         | `10 + 3`  | `13`       |
| `-`    | 减法         | `10 - 3`  | `7`        |
| `*`    | 乘法         | `10 * 3`  | `30`       |
| `/`    | 除法（浮点） | `10 / 3`  | `3.333...` |
| `//`   | 整除（取整） | `10 // 3` | `3`        |
| `%`    | 取模（余数） | `10 % 3`  | `1`        |
| `**`   | 幂运算       | `10 ** 3` | `1000`     |

**注意**：`/` 除法总是返回浮点数，`//` 整除返回整数（向下取整）

### 赋值运算符

用于给变量赋值，可以与其他运算符组合。

| 运算符 | 示例      | 等价于       |
| :----- | :-------- | :----------- |
| `=`    | `x = 5`   | `x = 5`      |
| `+=`   | `x += 3`  | `x = x + 3`  |
| `-=`   | `x -= 3`  | `x = x - 3`  |
| `*=`   | `x *= 3`  | `x = x * 3`  |
| `/=`   | `x /= 3`  | `x = x / 3`  |
| `//=`  | `x //= 3` | `x = x // 3` |
| `%=`   | `x %= 3`  | `x = x % 3`  |
| `**=`  | `x **= 3` | `x = x ** 3` |

### 比较运算符

用于比较两个值，返回布尔值 `True` 或 `False`。

| 运算符 | 名称     | 示例     | 结果    |
| :----- | :------- | :------- | :------ |
| `==`   | 等于     | `5 == 3` | `False` |
| `!=`   | 不等于   | `5 != 3` | `True`  |
| `>`    | 大于     | `5 > 3`  | `True`  |
| `<`    | 小于     | `5 < 3`  | `False` |
| `>=`   | 大于等于 | `5 >= 5` | `True`  |
| `<=`   | 小于等于 | `5 <= 3` | `False` |

### 逻辑运算符

用于组合布尔值，返回布尔结果。

| 运算符 | 名称 | 规则           | 示例             | 结果    |
| :----- | :--- | :------------- | :--------------- | :------ |
| `and`  | 与   | 两边都为真才真 | `True and False` | `False` |
| `or`   | 或   | 一边为真即为真 | `True or False`  | `True`  |
| `not`  | 非   | 取反           | `not True`       | `False` |

**短路求值**

逻辑运算符会"短路"：一旦结果确定，后续表达式不再计算。

- `and`：左边为 `False`，右边不再执行
- `or`：左边为 `True`，右边不再执行

### 运算符优先级

表达式中包含多个运算符时，按优先级顺序计算。

| 优先级    | 运算符                           | 说明           |
| :-------- | :------------------------------- | :------------- |
| 1（最高） | `()`                             | 括号           |
| 2         | `**`                             | 幂运算         |
| 3         | `+x`、`-x`、`not x`              | 正、负、逻辑非 |
| 4         | `*`、`/`、`//`、`%`              | 乘除类         |
| 5         | `+`、`-`                         | 加减类         |
| 6         | `==`、`!=`、`>`、`<`、`>=`、`<=` | 比较           |
| 7         | `not`                            | 逻辑非         |
| 8         | `and`                            | 逻辑与         |
| 9（最低） | `or`                             | 逻辑或         |

**建议：不确定优先级时用括号明确**

### 成员运算符

用于检查一个值是否存在于序列（字符串、列表等）中。

| 运算符   | 名称   | 示例               | 结果   |
| :------- | :----- | :----------------- | :----- |
| `in`     | 存在   | `"a" in "abc"`     | `True` |
| `not in` | 不存在 | `"d" not in "abc"` | `True` |

例如：

```python
# 实际应用：用户登录校验
username = input("请输入用户名：")
blacklist = ["admin", "root", "test"]

if username in blacklist:
    print("用户名已被禁用")
else:
    print("用户名可用")
```

### 身份运算符

用于比较两个变量是否指向**同一个对象**（而非值相等）。

| 运算符   | 名称 | 说明                 |
| :------- | :--- | :------------------- |
| `is`     | 是   | 两个变量引用同一对象 |
| `is not` | 不是 | 引用不同对象         |

例如：

```python
# 常见用法：检查 None
x = None
if x is None:
    print("x 是空值")
```

**注意**：`==` 比较值是否相等，`is` 比较对象id。日常开发多用 `==`。



## 字符串(String)

Python中的字符串用单引号 `'` 或双引号 `"` 括起来，同时使用反斜杠 `\` 转义特殊字符。

字符串是 Python 中最常用的数据类型之一。

### 索引

字符串是一个**字符序列**，可以通过索引访问每个字符。

python字符串有正向索引和反向索引。

假设字符串由6个字符组成，那么python字符串的索引范围是从 -6 到 5 。

```python
text = "Python"

# 正向索引：从 0 开始
# P y t h o n
# 0 1 2 3 4 5

print(text[0])   # P
print(text[1])   # y
print(text[4])   # o

# 负向索引：从 -1 开始（从右向左）
# P  y  t  h  o  n
# -6 -5 -4 -3 -2 -1

print(text[-1])  # n（最后一个字符）
print(text[-2])  # o
print(text[-6])  # P
```

### 切片

基本语法：`[start:end:step]` 

- `start`：起始索引（包含）
- `end`：结束索引（**不包含**）
- `step`：步长（默认为1）

```python
text = "Python Programming"

# 基本切片
print(text[0:6])     # Python（索引0-5）
print(text[7:11])    # Prog（索引7-10）
print(text[:6])      # Python（省略start，从0开始）
print(text[7:])      # Programming（省略end，到末尾）
print(text[:])       # 整个字符串（复制）

# 步长切片
print(text[::2])     # Pto rgamn（每隔一个取一个）
print(text[::-1])    # gnimmargorP nohtyP（反转字符串）

# 索引越界处理：Python 会自动处理
print(text[0:100])   # 超出长度，不报错，返回全部
```

### 字符串常用操作方法

#### 大小写转换

| 方法           | 说明               | 示例                                      |
| :------------- | :----------------- | :---------------------------------------- |
| `upper()`      | 全部转大写         | `"hello".upper()` → `"HELLO"`             |
| `lower()`      | 全部转小写         | `"HELLO".lower()` → `"hello"`             |
| `capitalize()` | 首字母大写         | `"hello".capitalize()` → `"Hello"`        |
| `title()`      | 每个单词首字母大写 | `"hello world".title()` → `"Hello World"` |
| `swapcase()`   | 大小写互换         | `"Hello".swapcase()` → `"hELLO"`          |

#### 查找与替换

| 方法                | 说明                                  | 示例                                    |
| :------------------ | :------------------------------------ | :-------------------------------------- |
| `find(sub)`         | 返回子串首次出现的索引，找不到返回 -1 | `"hello".find("l")` → 2                 |
| `index(sub)`        | 同 find，但找不到会报错               | `"hello".index("l")` → 2                |
| `rfind(sub)`        | 从右查找，返回索引                    | `"hello".rfind("l")` → 3                |
| `count(sub)`        | 统计子串出现次数                      | `"hello".count("l")` → 2                |
| `replace(old, new)` | 替换子串                              | `"hello".replace("l", "x")` → `"hexxo"` |

#### 去除空白字符

| `lstrip()` | 去除左侧空白 | `" hi ".lstrip()` → `"hi "` |
| `rstrip()` | 去除右侧空白 | `" hi ".rstrip()` → `" hi"` |

额外用法：

```python
text = "###Hello###"

```

### 字符串格式化

字符串格式化就是把变量插入到字符串中，拼成一个完整的句子。

**字符串格式化 = 在字符串里留"空位"，然后把变量填进去**

常见的字符串格式化方式：

| 方式        | 写法                    | 示例                    | 推荐度           |
| :---------- | :---------------------- | :---------------------- | :--------------- |
| f-string    | `f"文本{变量}"`         | `f"我叫{name}"`         | ⭐⭐⭐ 最推荐       |
| `.format()` | `"文本{}".format(变量)` | `"我叫{}".format(name)` | ⭐⭐ 兼容性好      |
| `%` 格式化  | `"文本%s" % 变量`       | `"我叫%s" % name`       | ⭐ 老旧，了解即可 |

例子：

```python
name = "小明"
age = 18

# 非格式化方式：用 + 拼接（繁琐）
print("我叫" + name + "，今年" + str(age) + "岁")

# 格式化方式1. f-string（Python 3.6+，最直观）
print(f"我叫{name}，今年{age}岁")

# 格式化方式2. format() 方法
print("我叫{}，今年{}岁".format(name, age))

# 格式化方式3. % 格式化（旧式，不推荐）
print("我叫%s，今年%d岁" % (name, age))

# 三种输出完全一样：
# 我叫小明，今年18岁
```

还有一些高阶用法，例如 **格式控制**（比如保留两位小数、左对齐右对齐），需要时候再查找。

## 控制流-条件判断

条件判断让程序能够根据不同的情况执行不同的代码，这是程序具备"智能"的基础。

### if 语句的三种形式

#### 1. 单一判断：`if`

#### 2. 二选一：`if-else`

```python
if score >= 60:

```

#### 3. 多分支：`if-elif-else`

## 控制流-循环

循环让程序能够重复执行某段代码，是编程中最重要的结构之一。

### while 循环

**基本语法：**

```python
while 条件:
    条件为真时重复执行的代码块
```

**执行流程图解：**

```python
     ┌─────────────┐
     │  判断条件   │
     └──────┬──────┘
            │
       ┌────┴────┐
       │ 条件为真？│
       └────┬────┘
       是 ↙     ↘ 否
    ┌──────┐   ┌──────┐
    │执行代码│   │ 结束 │
    └───┬──┘   └──────┘
        │
        └──→ 回到判断条件
```

**简单示例：**

```python
# 计算 1+2+3+...+100
total = 0
num = 1

while num <= 100:
    total += num
    num += 1

print(f"1到100的和：{total}")   # 5050
```

### for 循环

`for` 循环用于**遍历**可迭代对象（字符串、列表、范围等）。

**基本语法：**

```python
for 变量 in 可迭代对象:
    对每个元素执行的代码
```

**例子：**

例1.使用 `range(start, stop, step)` ：

```
# 倒序，
for i in range(5, 0, -1):
    print(i)
# 5 4 3 2 1
```

例2.遍历列表：

```python
fruits = ["苹果", "香蕉", "橙子"]

for fruit in fruits:
    print(f"我喜欢吃{fruit}")

# 输出：
# 我喜欢吃苹果
# 我喜欢吃香蕉
# 我喜欢吃橙子
```

例3.使用 `enumerate`，作用是在遍历可迭代对象（如列表、字符串等）时，同时获取元素的索引和值。

```python
for i, fruit in enumerate(fruits):
    print(f"索引{i}: {fruit}")
# 输出：索引0: 苹果  索引1: 香蕉  索引2: 橙子
```

### 循环控制语句

| 语句       | 作用                         |
| :--------- | :--------------------------- |
| `break`    | **立即跳出**整个循环         |
| `continue` | **跳过本次**循环，进入下一次 |

**break 示例：**

```python
# 查找第一个偶数
numbers = [1, 3, 5, 7, 8, 9, 10]

for num in numbers:
    if num % 2 == 0:
        print(f"找到第一个偶数：{num}")
        break   # 找到后立即退出

# 输出：找到第一个偶数：8
```

**continue 示例：**

```python
# 打印 1-10 中的所有奇数（跳过偶数）
for i in range(1, 11):
    if i % 2 == 0:
        continue   # 跳过本次循环
    print(i, end=" ")

# 输出：1 3 5 7 9
```

### 与 `else` 的搭配用法

在 Python 中，`else` 可以和 **`for`**、**`while`** 搭配使用（Python 独有）。

| 搭配         | 语法                        | 执行时机                           |
| ------------ | --------------------------- | ---------------------------------- |
| `for-else`   | `for ...: ... else: ...`    | 循环**没有被 break 打断**时执行    |
| `while-else` | `while 条件: ... else: ...` | 循环**正常结束**（条件变假）时执行 |

#### `for-else` 详解

**基本语法：**

```
# 基本结构
for 变量 in 可迭代对象:
    if 某个条件:
        break    # 提前退出
else:
    # 循环没有被 break 打断时执行
    执行代码
```

**例子：**

检查列表中是否有偶数

```python
numbers = [1, 3, 5, 7, 9]

for num in numbers:
    if num % 2 == 0:
        print(f"找到偶数：{num}")
        break
else:
    print("列表中没有偶数")

# 输出：列表中没有偶数
```

逻辑： 

- 如果找到偶数 → `break` 跳出 → `else` 不执行
- 如果没找到偶数 → 循环正常结束 → `else` 执行

### `while-else` 详解

如果循环没有被 `break` 打断，就执行 `else` 

**基本语法：**

```python
# 基本结构
while 条件:
    if 某个条件:
        break
else:
    # 循环条件变假而正常结束时执行
    执行代码
```

例子：

```python
attempts = 3

while attempts > 0:
    pwd = input("请输入密码：")
    if pwd == "123":
        print("登录成功")
        break
    attempts -= 1
    print(f"还剩 {attempts} 次机会")
else:
    print("次数用完，账户锁定")   # 3次都错才执行

# 如果中途输对了 → break → else不执行
# 3次全错 → 循环正常结束 → else执行
```
