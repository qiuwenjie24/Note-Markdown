# Zotero

## 一、Zotero 简介

Zotero 是一款 **免费、开源** 的文献管理软件，支持 **网页抓取、PDF管理、参考文献生成** 等功能，适合学生、科研人员和学术工作者。

以下是基于 7.0.15 (64-bit) 版本进行介绍。

## 二、下载与安装

1. **下载地址**：

   官网下载 **Zotero 客户端**：https://www.zotero.org/

2. **安装浏览器插件**：

   在官网中安装Zotero浏览器插件。

3. **安装客户端**：

   运行下载的安装包（Windows/macOS/Linux），按提示完成安装。

4. **关联客户端与插件**：

   打开Zotero客户端，进入 `工具` → `设置` → `同步`，登录Zotero账号（需注册）。


## 三 、Zotero 界面认识

打开 Zotero，你会看到三个主要区域：

- **左侧栏**：`我的文库` 存储你添加的所有文献，下面的 子文件夹(集合) 则是文献的分类。
- **中间栏**：当前 文件夹(集合) 下的文献列表。
- **右侧栏**：当前选中文献的详细信息（元数据、笔记、标签等）。

## 推荐设置

**自动重命名 PDF：**

编辑 → 设置 → 常规 → 勾选`重命名链接的文件` → 点击`自定义文件名格式`：自定义文件名模板。

(似乎只对后添加的文献生效)



## 四、基础使用教程

**添加文献：**

方法1：直接拖 PDF 到 Zotero 的中间区域，Zotero 会自动尝试识别元数据（如标题、作者、期刊、年份等）。

方法2：点击上方工具栏的魔棒图标（“添加条目”），输入 DOI 或 arXiv ID，Zotero 会自动下载并添加完整的文献信息。

方法3：如果安装了Zotero 的浏览器插件，点击浏览器工具栏的 Zotero 图标，Zotero 会将浏览器中打开的文献下载并保存到你的库中。

> 注意：对于自己写的PDF文件，Zotero可能无法识别文件类型，默认识别为附件，这导致没有独立的元数据（如作者、年份、标题等），无法添加修改作者、年份等信息。
>
> 解决方式：创建空条目--->填写信息--->拖入文件，成为该条目附件。



**分类文献：**

在左侧栏右键点击`我的文库`，选择`新建分类`，创建 文件夹。

你可以把一篇文献放到多个 文件夹(集合) 中，不影响原始文献的位置（类似标签）。



**阅读和做笔记：**

1. 双击 PDF 可以在 Zotero 内部打开阅读器。

2. 可直接高亮、添加注释。

3. 注释会自动同步为“笔记”出现在右侧栏（Notes 标签页）。

4. 你也可以手动添加“笔记”，用于记录想法或总结。

## 五、Zotero 管理文献原理

当添加文献时，Ztero 会另外在一个专门的目录（默认是 `Zotero/storage/随机文件夹名/`）中生成该文献的 PDF 文件等附件。而文献信息（作者、标题、年份、DOI 等）是存储在 数据库 中（不是文件系统的形式）。

在`我的文库`中看到的“一个条目”，其实对应的是：**一条文献元数据 + 附件链接**。

你把同一篇文献放入不同的 文件夹(集合) ，相当于是在该 文件夹(集合) 中引用了它。在该 文件夹 中删除文献 只是把它从该 文件夹 中移除，文献的PDF文件不会被删除。

如果你在`我的文库`中删除文献，则相当于真的删除，也会删除目录下的PDF文献。

如果你将一个文献添加两次，则相当于有两个独立副本，目录中会生成两个对应的PDF附件，删除其中一个，不影响另一份。

## 优势

相比于手动建文件夹管理文献的优势：

1. PDF 文件命名统一化，比如可自动重命名为“作者 - 标题 - 年份”。
2. 自动识别文献信息，比如作者、DOI、期刊等。
3. 支持高级筛选、标签等方式快速搜索目标文献或文献内容。
4. 可以将一个文献归入多个 文件夹(集合) 。
5. 一键导出`.bib`文件，实现文献引用自动化。
6. 支持云同步。
