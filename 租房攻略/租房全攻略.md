# 租房全攻略

## 一、租房时机

- **租房旺季** (2-3月，6-7月)：租金可能上涨10%-20%，房源紧俏。
- **租房淡季** (11-12月)：议价空间较大，容易找到性价比高的房子。
- **预算红线**：月租金**不要超过到手工资的30%**。

## 二、租房全流程【5步走】

1. **确定要求** → 2. **网上找房** → 3. **线下看房** → 4. **签约** → 5. **入住**

### 第一步：确定租房要求

- **通勤范围**：由通勤时间决定，单程**建议在30分钟之内，尽量不要超过1小时**，优先考虑靠近地铁或公交站。
- **预算**：月租金**建议不要超过到手工资的30%**。
- **户型偏好**：
  - **整租**：独立空间，价格最高。
  - **合租**：共用公共区域，性价比高。
  - **品牌公寓**：服务好，省心，但每个月需交额外的服务费。
- **其他需要**：是否允许养宠物、是否有阳台、是否可以做饭等。

### 第二步：网上找房（渠道与甄别）

#### 1. 找房渠道对比

| 渠道类型     | 优点                 | 缺点                                  | 代表平台                       |
| :----------- | :------------------- | :------------------------------------ | :----------------------------- |
| **品牌公寓** | 省心、服务好、家具全 | 价格偏高，每月额外收服务费（房租10%） | 自如、泊寓                     |
| **中介平台** | 房源真实性相对较高   | 需支付中介费（通常为0.5-1个月房租）   | 贝壳、链家                     |
| **个人直租** | 价格可谈，无中介费   | 需自己筛选，真实性参差不齐            | 豆瓣小组、优区生活、小区公告栏 |

服务费（以自如平台为例）包括一下服务：
1.租期内的宽带网络
2.整租每个月一次全屋保洁/合租每个月两次公区保洁
3.屋子里所有电器家具免费维修
4.保险（人身财产安全险）
5.管家服务（协调室友，物业邻居，代办门禁卡等）
6.客服服务（办理租赁登记备案，开发票等）

#### 2. 角色定义与注意

- **房东直租**：最理想，无中介费，直接与产权人签约。
- **二房东**：转租原房东的房子。**坑比较多，不推荐新手选择**。如必须租，务必看到原合同中“允许转租”的条款。
- **中介**：撮合交易，收钱后对后续租房问题一般不负责。
- **平台管家**：平台员工，只负责推销本平台的房子，会对租房后的问题（如维修等）负责。签约是与公司（平台）签约。

注：平台的房子有的是平台自己的，有的代托管的，因此房屋类型很多种，酒店公寓、居民楼之类的。

#### 3. 线上需要了解的信息

- **房屋基础信息**：建成/装修时间、楼层、面积、布局、家具清单、是否有电梯。
- **全部费用清单**：
  - 租金、押金（是押一付一还是押一付三？）
  - 中介费（如果是中介渠道）
  - 物业费、网费、水电燃气费（民用还是商用？）
- **上海每月费用参考**：
  - 物业费：50-300元
  - 水电煤：100-200元
  - 网费：50-100元
  - **民用水电价格**：电 0.617元/度，水 4.05元/吨
  - **商用水电参考价**：电 0.8-1.3元/度，冷水 5-7元/吨，热水 20-60元/吨
- **注意事项**
  - 低于同小区均价30%的房源，通常有各种坑，需极度谨慎核实，一般建议直接pass。
  - 要求先付定金或看房费的，直接pass。
  - 家具、床垫保护膜没撕，且装修看起来比较新的，可能是刚装修没多久的房子，有甲醛风险。

### 第三步：线下看房

#### 看房核心检查清单

| 检查类别       | 具体检查项目                                                 |
| :------------- | :----------------------------------------------------------- |
| **家电家具**   | **开机测试**：空调、冰箱、洗衣机、热水器、油烟机等是否工作正常。 **看能耗**：空调、冰箱等大电器，能效等级最好为1-3级，高于3级（4或5级）或没有标识的会很费电，需特别注意。 |
| **水电设施**   | **水压**：打开所有水龙头，测试水流大小。 **下水**：马桶多冲几次，洗漱池、厨房水槽下水是否通畅。 **电表**：确认电表在屋内（可能是房东自装）还是楼栋（国家电网）。 **燃气**：如有燃气灶，点火测试，并问清燃气费缴纳方式。 |
| **房屋状况**   | **防潮防漏**：检查墙角、柜子、天花板是否有脱皮、发霉、水渍。 **气味**：卫生间是否有下水道反味。 **隔音**：敲敲墙壁判断是否为隔断墙，关窗感受楼道和马路噪音。 **串串房警惕**：家具很新但无生活痕迹，或进屋有刺鼻气味，可能是串串房（装修劣质有害物质超标）。 |
| **采光通风**   | 窗户大小、朝向（朝南采光最佳）。                             |
| **周边与环境** | **便利**：离地铁/公交站多远？有无超市、药店？ **安全**：小区是否允许外人随意进出、小区是否有监控、低楼层的阳台有无围栏。 **卫生**：楼道是否干净。 |
| **拍照取证**   | **看房时全程拍照/录像**，特别是房屋的瑕疵（如墙面破损、家具划痕），回去后再仔细判断，**切勿当场做决定**。 |
| **合租**       | **问清楚**：室友性别、职业、作息（是否夜班）、是否养宠物、公共区域卫生怎么分配。通过垃圾桶、冰箱等反映是由的生活习惯。 |

**串串房的特征**：

- 进门刺鼻味道很大
- 家具都很新，但都是杂牌便宜货，能耗高
- 没有生活过的痕迹（可以询问装修的时间）
- 装修材质都是劣质品

**注意事项**：

- 避免一楼和顶楼。一楼偏潮湿，采光不好，蚊虫多；顶楼冬冷夏热，天花板易漏水。
- 底楼做餐饮的不建议租，容易招惹虫子。
- 有时间可以早晚看一次房子，白天看采光，晚上听噪音。

**额外补充**（自己感受）：

- 尽量不要选靠近马路的，虽然可能觉得自己可以接受，但是当你睡眠不好、状态不好，需要极度安静的环境的时候，就会发现很难受。
- 没有电梯，需要爬楼梯的直接pass。搬水、搬重物比较麻烦，经常喜欢来回出去溜达散步也很麻烦。
- 多给几天自己感受房子周围的环境，买东西是否便利，出去玩交通是否便利，附近吃的是否多，是否有散步、跑步的地方等。
- 亲自体验一下通勤时间，以及考虑是否需要买载具（自行车或电车）。

### 第四步：签约合同（重中之重）

#### 1. 签约前确认

- **核实房东身份（必须看原件）**
  - **房东**：身份证 + 房产证（名字一致）。
  - **二房东**：身份证 + 与原房东的租赁合同（含“允许转租”条款）+ 原房东的身份证及房产证复印件。
  - **中介/管家**：确认其公司是否正规备案。
- **抄表记录**：与房东一起核对并拍照记录**水、电、燃气表的当前底数**，写在交接单上双方签字。
- **明确关键问题**：
  - 维修责任：正常损耗（如灯泡坏了、空调不制冷）谁来修？费用谁出？
  - 费用缴纳：水电燃气是自己交还是房东代缴？（代缴可能有猫腻）
  - 特殊规定：能否养宠物？能否在墙上贴纸或打孔？
  - 留存凭证：拍下的房屋现状视频/照片，发给房东确认，避免退租时被扣钱。如果有已经坏的家具等，确认什么时候修好。

#### 2. 砍价策略（待完善）

- **对房东/中介**：以“长租、稳定、爱惜房子”为理由，或指出房屋的瑕疵（如交通、设施老旧）进行砍价。
- **对平台管家**：租金标价是系统定价，**基本不能砍**。可以尝试问：“有没有免服务费的优惠活动？”

#### 3. 合同条款核对表（口头承诺必须写进合同）

| 合同条款       | 必须明确的细节                                               |
| :------------- | :----------------------------------------------------------- |
| **租期**       | 写具体日期（如2025年5月10日到2026年5月9日），而非“一年”。    |
| **租金与支付** | 每月几号前交？转账到哪个账户？                               |
| **押金**       | 押几个月？退租后几个工作日内返还？什么情况会扣押金？         |
| **各项费用**   | 水电燃气网费单价多少？物业费、清洁费、门禁费等由谁承担？     |
| **维修责任**   | 家电家具坏了，谁负责联系维修？几天内修好？如超时未修，租客有权自行维修并从下月房租中扣除费用。 |
| **退租与转租** | 提前退租扣多少违约金？合同到期需提前多久通知？中途是否允许转租？ |
| **其他**       | 和房东的一些约定或承诺                                       |

#### 4. 签约步骤与备案

1. **顺序**：先签《房屋租赁合同》→ 再签《居间服务合同》（中介合同，如有中介）→ 最后支付中介费，并索要**收据或发票**。
2. **备案**：合同签完后**30天内**，要求或自行办理**租赁合同登记备案**。这是房东的法定义务，也是你后续办理**居住证**的必要前提。
3. **留凭证**：所有转账备注“XX月租金+押金”，妥善保管合同、收据、沟通记录。

注：

- 和平台签约一般是在平台的APP上签电子合同。
- 最好不要一开始签一年及以上的长约，万一住得不舒服也容易换。
- 尽量选择押一付一。

### 第五步：入住

- 恭喜你找到合适的房子！
- 如果合同没明确禁止，建议换锁芯（几十块钱），退租时换回去即可。
- 自如等品牌公寓一般是密码锁，可以自己改一个临时密码。
- 确认所有家电功能完好。
- 保留好房东、物业、紧急维修的联系方式。

**入住前需要购买或准备的东西**：

- 床上用品：枕头、被子、床单、床垫
- 洗漱用品：毛巾、牙刷、牙膏、杯子、沐浴露、洗发水、护发素、香皂、洗衣液，洗衣桶、洗脸盆。
- 清洁工具：拖把、扫把、马桶刷、洁厕灵、抹布
